# 11_RISK_COMPLIANCE：风险与合规说明

## 投资风险

本项目生成的是公开信息整理和研究辅助材料，不构成投资建议。

报告中不得出现：

- 必买；
- 必涨；
- 卖出；
- 满仓；
- 保证收益；
- 无风险套利；
- 明确下单指令。

## 数据风险

公开数据源可能存在：

- 接口变更；
- 字段变更；
- 延迟；
- 缺失；
- 限流；
- 风控；
- 错误返回；
- 非交易日空数据。

所有报告必须包含数据质量说明。

## 第三方项目使用风险

`a-stock-data` 是第三方开源项目。本项目使用它时需要：

- 记录来源；
- 查看 LICENSE；
- 保留必要 attribution；
- 不整包复制；
- 不声称自己是数据源官方；
- 不绕过任何鉴权。

## 网络请求风险

所有请求必须：

- 设置 timeout；
- 限制数量；
- 不并发刷取；
- 不无限重试；
- 出错写入日志。

## LLM 风险

如果后续接入 LLM，需要注意：

- LLM 可能幻觉；
- LLM 不能作为数据源；
- LLM 输出必须基于 Evidence Pack；
- 不能让 LLM 猜测未匹配股票；
- 报告中必须标注推测。

## 合规边界声明模板

建议每份报告末尾包含：

```text
本报告由公开数据源自动整理生成，仅用于信息整理和研究辅助，不构成任何投资建议、交易建议或收益承诺。数据可能存在延迟、缺失或错误，请以官方公告和交易所披露为准。
```
