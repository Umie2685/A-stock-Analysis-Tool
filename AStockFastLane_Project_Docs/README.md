# AStockFastLane 项目启动文档包

本压缩包用于启动一个全新的 A 股热点分析工具项目。新项目以 `a-stock-data` Skill / README / SKILL.md 中的数据源接口思路为核心参考，目标是快速完成一个可运行的“自动抓新闻 / 公告 / 研报 / 行情 → 结构化证据包 → 报告”的工具雏形。

## 角色分工

- 用户：项目协调者，负责确认目标、运行 Codex、粘贴交付结果、决定是否继续下一步。
- ChatGPT：项目经理 / 架构把关 / Codex 任务拆解 / 验收官。
- Codex：执行具体代码开发、探针、文档更新和测试。

## 使用方式

1. 新建一个空项目目录，例如：`AStockFastLane/`。
2. 将本压缩包解压到新项目的 `docs/` 或项目根目录。
3. 先阅读：`00_START_HERE.md`。
4. 把 `09_CODEX_TASKS_MVP0.md` 中的第一个任务发给 Codex。
5. Codex 完成后，把它的修改文件、运行结果、报错和生成文件贴回给 ChatGPT 验收。

## 核心原则

- 快速开发，但不无序开发。
- 每次只让 Codex 做一个小任务。
- 先 probe，再 provider，再 pack，再 report。
- 不直接整包复制第三方项目。
- 不做自动交易。
- 不把未经验证的数据直接用于投资建议。
